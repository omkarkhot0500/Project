# Installation and Setup

- Fork the repo and clone it
  ```bash
  git clone
  ```

## Set up the client side

```bash
cd frontend
npm install
npm run dev
```

## Docker Set up

# 1. Clone the project

git clone
cd frontend

# 2. Build the Docker image and run the container

docker-compose up --build

# The app will be available at http://localhost:3000
